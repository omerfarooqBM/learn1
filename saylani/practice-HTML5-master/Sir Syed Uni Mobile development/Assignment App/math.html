Q.3:

Do the following using JS Mathematic Expressions.

  1. Declare a variable.
  2. Show the value of variable in your browser like “Value after variable declaration is: ??”.
  3. Initialize the variable with some number.
  4. Show the value of variable in your browser like “Initial value: 3”.
  5. Increment the variable.
  6. Show the value of variable in your browser like “Value after increment is: 4”.
  7. Add 7 to the variable.
  8. Show the value of variable in your browser like “Value after addition is: 11”.
  9. Decrement the variable.
  10. Show the value of variable in your browser like “Value after decrement is: 10”.
  11. Show the remainder after dividing the variable’s value by 3. Output : “The remainder is : 1”

A:
var decVari;
document.write("Value after variable declaration is: "+ decVari);
decVari = 3;
document.write("Initial value: "+ decVari);
decVari = ++decVari;
document.write("Value after increment is: "+ decVari);
decVari = decVari + 7;
document.write("Value after addition is: "+ decVari);
decVari = --decVari;
document.write("Value after decrement is: "+ decVari);
decVari= decVari % 3;
document.write("The remainder is : "+ decVari);

  4. Convert it to Celsius & output “NNoF is NNoC”.

var temCelsius = +prompt("Please Enter Celsius Temprature: ","Number");
var temInFahrenheit = (temCelsius * 9/5) + 32;
document.write(temCelsius+ " ℃ " +" is " +temInFahrenheit+ " ℉ ");

var temFahrenheit = +prompt("Please Enter Fahrenheit Temprature: ","Number");
